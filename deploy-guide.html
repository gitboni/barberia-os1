20 minutos
1
Crear cuenta en Supabase

2
Ejecutar SQL de configuración
En el SQL Editor de Supabase, ejecuta este script completo:
-- 1. TABLA DE BARBERÍAS (multi-tenant)
CREATE TABLE shops (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  nombre TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  whatsapp TEXT,
  plan TEXT DEFAULT 'free',
  webhook_url TEXT,
  duracion_turno INTEGER DEFAULT 45,
  activo BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 2. TABLA DE CITAS (ya la tienes, agregar shop_id)
ALTER TABLE citas 
  ADD COLUMN IF NOT EXISTS shop_id UUID REFERENCES shops(id),
  ADD COLUMN IF NOT EXISTS created_at TIMESTAMPTZ DEFAULT NOW();

-- 3. TABLA DE CONVERSACIONES (ya la tienes, agregar shop_id)
ALTER TABLE conversaciones
  ADD COLUMN IF NOT EXISTS shop_id UUID REFERENCES shops(id);

-- 4. TABLA DE BARBEROS (nueva, reemplaza config estática)
CREATE TABLE barberos (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  shop_id UUID REFERENCES shops(id),
  nombre TEXT NOT NULL,
  activo BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 5. INSERT de prueba — tu barbería
INSERT INTO shops (nombre, slug, whatsapp, plan)
VALUES ('Elite Cuts', 'elite-cuts', '+18098498528', 'pro')
RETURNING id;
-- Guarda el ID que retorna — lo usas como SHOP_ID

-- 6. Índices para performance
CREATE INDEX idx_citas_shop_fecha ON citas(shop_id, fecha);
CREATE INDEX idx_conv_shop_chat ON conversaciones(shop_id, chat_id);
CREATE INDEX idx_barberos_shop ON barberos(shop_id);

Agregar función de carga real (reemplaza loadState)
// Función que carga citas REALES desde Supabase
async function loadCitasSupabase() {
  const hoy = new Date().toISOString().split('T')[0];
  try {
    const res = await fetch(
      `${SUPABASE_URL}/rest/v1/citas` +
      `?shop_id=eq.${SHOP_ID}` +
      `&fecha=eq.${hoy}` +
      `&estado=neq.Cancelada` +
      `&order=hora.asc`,
      {
        headers: {
          'apikey': SUPABASE_KEY,
          'Authorization': `Bearer ${SUPABASE_KEY}`
        }
      }
    );
    const citas = await res.json();
    
    // Mapear al formato del state existente
    state.clients = citas.map(c => ({
      name: (c.nombre || 'Cliente').toUpperCase(),
      time: (c.hora || '00:00').substring(0, 5),
      telefono: c.telefono || '',
      servicio: (c.servicio || 'CORTE CLÁSICO').toUpperCase(),
      barbero: (c.barbero || 'cualquiera').toUpperCase(),
      status: 'wait'
    }));
    
    updateStatuses();
    render();
    console.log(`✓ ${citas.length} citas cargadas desde Supabase`);
  } catch(e) {
    console.error('Error cargando citas:', e);
    showToast('⚠ Error conectando con la base de datos', 'error');
  }
}

// Auto-refresh cada 30 segundos
loadCitasSupabase();
setInterval(loadCitasSupabase, 30000);

{
  "rewrites": [

    { "source": "/admin/:shop", "destination": "/admin.html" },
    { "source": "/registro", "destination": "/register.html" }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        { "key": "Access-Control-Allow-Origin", "value": "*" },
        { "key": "X-Frame-Options", "value": "SAMEORIGIN" }
      ]
    }
  ]
}

Instalar en tablet Android como PWA / APK
5 minutos
A
Opción A — PWA (recomendada, cero costo)
1. Abre Chrome en la tablet Android
2. Ve a tu-proyecto.vercel.app/display.html
3. Menú (⋮) → "Agregar a pantalla de inicio"
4. Se instala como app nativa con ícono
5. Abre → pantalla completa, sin barra de Chrome
6. ✓ Funciona aunque no haya internet (citas en caché)
B
Opción B — APK con WebIntoApp (gratis, 10 min)
1. Ve a webintoapp.com
2. URL: tu-proyecto.vercel.app/display.html
3. Nombre: AuraOS · Orientación: Landscape
4. Descarga APK → instala en tablet
5. Distribuye por WhatsApp a tus clientes barberías
→ webintoapp.com

// En display.html — al inicio del script:
const shopSlug = new URLSearchParams(location.search).get('shop')
               || location.pathname.split('/').pop()
               || 'elite-cuts';

// Cargar shop_id desde Supabase usando el slug:
async function loadShop() {
  const res = await fetch(
    `${SUPABASE_URL}/rest/v1/shops?slug=eq.${shopSlug}&select=id,nombre`,
    { headers: { 'apikey': SUPABASE_KEY }}
  );
  const [shop] = await res.json();
  if(shop) {
    SHOP_ID = shop.id;
    document.getElementById('shop-name').textContent = shop.nombre;
    await loadCitasSupabase();
  }
}
